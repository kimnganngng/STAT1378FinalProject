Introduction

For this report, I will analyse the beverage nutrition from Starbucks with the ‘starbucks’ and ‘starbucks_raw’ dataset. Both two data set included the information about drinks’ names, size and other nutritions facts.

This report is created to answer two research questions mentioned below:

  1. Is there a linear relationship between calories and total fat?
  2. Are there any correlation between all nutritions’ values?

Methods

  1. Linear relationship between calories and total fat:
    The linear relationship between calories and total fat can be measured by estimating and testing \(\beta\) in the linear regression model. In which, \(calories = \alpha + \beta total fat + \epsilon\), where \(\epsilon \sim N(\mu=0, \sigma^2 = 1)\).

  2. Correlation between all nutritions’ values
    For this question, I used heat mapping to visualize the correlation of all nutrition attributes and write an conclusion based on that graph. The method using to export this graph is Kendall Rank Correlation and plot.matrix (Klinke 2022) package.
